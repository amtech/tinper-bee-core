# bee-core
tinper-bee组件库的核心样式及公用方法库

## 核心样式
放在scss文件夹下，由index.scss统一导出
包括
- 重置浏览器样式
- 全局样式设置
- 图标样式
- 波纹效果、阴影样式、文本及字体样式
- 主辅色及常用颜色设置
